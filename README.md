# PolytopiaGuide
A guide for the mobile game *The Battle for Polytopia*, formatted as a technical manual.

# Get started 

*The Battle for Polytopia* is an award-winning turn-based strategy game. 

*Polytopia* is a ‘4X’ (Explore, Expand, Exploit, Exterminate) type game, similar to the *Civilization* series, albeit with much more streamlined and fast-paced game mechanics and a brightly-coloured, low-polygon art style. 

Versions of *Polytopia* are available for iOS, Android, Windows, Mac, and Linux (via Steam), and Nintendo Switch. 

This unofficial guide uses screenshots from the Android and Switch versions of the game, but the strategies and tactics discussed here can be applied to any version. 

This guide has been written with beginners in mind. It explains the core concepts of *Polytopia*, from how to interact with the game, to a complete explanation of the combat mechanic. 

## How to download Polytopia – on iPhone, Android, PC, or Switch

1. Download Polytopia from:

* [iOS App Store](https://apps.apple.com/gb/app/the-battle-of-polytopia/id1006393168)
* [Google Play](https://play.google.com/store/apps/details?id=air.com.midjiwan.polytopia&hl=en_GB)
* [Steam](https://store.steampowered.com/app/874390/The_Battle_of_Polytopia/)
* [Epic Games](https://store.epicgames.com/en-US/p/the-battle-of-polytopia-12fed6)
* [Nintendo Store](https://www.nintendo.com/en-gb/Games/Nintendo-Switch-download-software/The-Battle-of-Polytopia-2273167.html)

2. Open the game

3. Click ‘New Game’ to start a single player game against computer-controlled opponents or Click ‘Multiplayer’ to set up a game against human opponents
